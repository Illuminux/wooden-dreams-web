## Pflegehinweise

1. Neue technische Zeichnungen nach Moeglichkeit immer als PDF/PNG-Paar ablegen.
2. Dateinamen konsistent und bauteilbezogen halten (`teil_funktion_variante_zeichnung`).
3. Varianten und Explosionsbilder klar mit Suffixen wie `_exp` oder `_zsb` markieren.
4. Bei neuen Dateien diese README aktualisieren, damit die Zuordnung nachvollziehbar bleibt.
