## Pflegehinweise

1. Neue technische Zeichnungen nach Moeglichkeit immer als PDF/PNG-Paar ablegen.

3. Varianten und Explosionsbilder klar mit Suffixen wie `_ex` kennzeichnen.
4. Bei neuen Dateien diese README aktualisieren, damit die Zuordnung nachvollziehbar bleibt.
